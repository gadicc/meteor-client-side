Option 1 - From a script tag (easiest)

Easiest, but you aren't guaranteed the same bundle each time unless you specify @= for all packages (including meteor-platform). If you do, browser will cache for 1 yr, otherwise 1 hr.

<script type='text/javascript' src='/bundle.min.js?release=METEOR@1.1.0.2&deps=meteor-platform,gadicohen:famous-views@1.2.0,etc'></script>
{{> generateBundle}}
